# 2026 H1 闪购述职报告｜Street Flight V2

这是述职报告的电影化飞行优化分支。重点不是增加更多动画，而是修复航线与镜头控制的结构问题。

## 本轮优化

- 将停靠点移入澳门城市与路氹街区，减少镜头面对海面或地图空白。
- 将原本会自折返的航线改为单向、连续、不自交的街道走廊。
- 取消“用停靠时望向建筑的方向作为下一段起飞方向”的错误逻辑。
- 将飞行器朝向与驾驶员镜头拆分：机体沿道路切线飞行，镜头只在最后接近时转向地标。
- 报告淡出期间立即开始移动，离场等待由 520ms 降为 180ms。
- 到达稳定时间由 420ms 降为 220ms，并加入极轻微悬停呼吸。
- 地标构图加入偏轴瞄准，使地标进入画面右下安全区，减少被左侧报告面板遮挡。
- 建筑退让走廊与实际飞行航线共用同一组坐标，降低穿楼概率。

## 运行

页面使用 ES Modules，必须通过 HTTP 服务打开：

```bash
python3 -m http.server 8080
```

打开：

```text
http://localhost:8080/reports/2026-h1-cinematic/
```

## 当前边界

这一版属于第一轮路线与镜头重构。航线仍是经过设计的电影化城市走廊，并非完整 OSM 路网 A* 导航。下一轮应在真实浏览器逐章录屏后，根据构图、碰撞审计及空白视野比例继续校准。
